# Teste

## Primeiros passos

**Faça um clone do projeto**

```bash
git clone https://github.com/henriquesouzadev/consulta-react.git
```

**Comandos**

```bash
# Instale as dependências do projeto
npm install

# Inicie o projeto em modo de desenvolvimento
npm start
```

Acesse `http://localhost:3000/`
